# Проект: SkillCinema

## Описание приложения
SkillCinema - мобильное приложение для поиска информации о фильмах.

## Основной функционал:
- Подборки фильмов
- Поиск фильмов
- Добавление в избранное
- Просмотр детальной информации о фильмах

## Тестовое покрытие:
- Функциональное тестирование
- UI/UX тестирование
- Тестирование на мобильных устройствах
